### Начало работы 
 * Настройте pipenv, почтитать можно здесь
  https://realpython.com/pipenv-guide/
 * По поводу работы с Gitом: 
 
 Качаем репозиторий    
 ```shell
 git clone git@github.com:grigagod/chat-example.git
 ```
 Переходим в него
```shell
 cd chat-example/ 
 ```
 Для справки можно посмотреть remote branches
```shell
branch -a  
 ```
 Для того чтобы получить локальную копию нужного branch,
 где вы будете работать(например client), пишем:
 ```shell
 checkout client
 ```
Теперь за работу!

PS: удобная комманда, чтобы увидеть последние изменения на текущей ветке.
```shell
git log -1 --stat
```
